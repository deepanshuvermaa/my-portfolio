A comprehensive Facebook ad scraping and AI-powered creative generation platform. Built using Python, Selenium, and BeautifulSoup to automatically scrape and compile large volumes of Facebook ad data. The platform includes an AI-assisted ad generation tool that allows users to select images/videos and generate creative variations using multiple AI models and image generation APIs. Deployed on Google Cloud Platform, this tool significantly increased organizational productivity and ROI by streamlining the creative testing process for performance marketers.
