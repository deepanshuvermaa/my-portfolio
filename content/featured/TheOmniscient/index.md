A Progressive Web App designed as a curated knowledge management platform with an ancient Japanese library aesthetic. Features 221 curated links across 11 knowledge domains, offline-capable functionality, link visit tracking, and personal note-taking. Built with React and Vite, implementing a minimalist touch-friendly interface with responsive design optimized for mobile and desktop. Includes PWA support with service worker, grid and list view toggles, and a color palette inspired by traditional Japanese manuscripts. Future integrations planned for Google Authentication and Google Sheets synchronization.
